# 🌐 Reflection – Using GitHub Pages

## STAR Method Reflection

---

### ⭐ Situation

I needed to publish my digital portfolio online so that it could be
accessed by anyone with the link. I had never deployed a website
before and was unfamiliar with GitHub Pages or how web hosting works.

---

### 🎯 Task

My task was to enable GitHub Pages on my portfolio repository, deploy
my Markdown files as a live website, and submit the public URL to
Teams as evidence of completion.

---

### ⚙️ Action

To complete this task, I:
- Navigated to my repository's Settings → Pages section
- Selected the `main` branch as the source
- Waited for GitHub to generate the live URL
- Tested the URL in my browser to confirm the site was live
- Added the live link to my portfolio README

---

### ✅ Result

My portfolio is now publicly accessible at:
`https://lefa07.github.io/`

This was a significant achievement because I deployed a real website
without needing a paid hosting service. GitHub Pages is a powerful
free tool that I will use in future projects. This experience gave me
a foundational understanding of web deployment and version control
that is directly relevant to my career.
